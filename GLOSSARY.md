# Glossary

**Version:** 4.0

This document provides a comprehensive glossary of terms, acronyms, and concepts used throughout the Freelancer Financial Hub documentation. It is intended for all audiences, from marketing to engineering.

| Term/Acronym | Definition |
| :--- | :--- |
| **AI** | Artificial Intelligence. |
| **API** | Application Programming Interface. |
| **CAC** | Customer Acquisition Cost. |
| **CI/CD** | Continuous Integration / Continuous Deployment. |
| **CRM** | Customer Relationship Management. |
| **DPIA** | Data Protection Impact Assessment. |
| **E2E** | End-to-End (testing). |
| **ERD** | Entity-Relationship Diagram. |
| **FFH** | Freelancer Financial Hub (the official acronym for the product). |
| **GDPR** | General Data Protection Regulation. |
| **JTBD** | Jobs to Be Done. A framework for understanding customer needs. |
| **KPI** | Key Performance Indicator. |
| **LLM** | Large Language Model. |
| **LTV** | Customer Lifetime Value. |
| **MFA** | Multi-Factor Authentication. |
| **MRR** | Monthly Recurring Revenue. |
| **MVP** | Minimum Viable Product. |
| **OCR** | Optical Character Recognition. The technology used to scan receipts. |
| **PCI-DSS** | Payment Card Industry Data Security Standard. |
| **PII** | Personally Identifiable Information. |
| **PRD** | Product Requirements Document. |
| **RLS** | Row-Level Security. A critical database security feature. |
| **SaaS** | Software as a Service. |
| **SOC 2** | Service Organization Control 2. A security compliance framework. |
| **SOP** | Standard Operating Procedure. |
| **STRIDE** | A threat modeling framework (Spoofing, Tampering, Repudiation, Information Disclosure, Denial of Service, Elevation of Privilege). |
| **UI** | User Interface. |
| **UX** | User Experience. |
